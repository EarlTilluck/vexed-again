How To Play Moving Blocks
  • Move blocks left or right by swiping them in that direction.
  • Blocks fall down due to gravity if there is an empty space below them.
  • Move similar blocks next to each other to make them disappear.
  • You win when all blocks disappear.
 Scoring

Each level has a 'Par' score which denotes how may moves it ideally takes to pass a level. Try to solve the level with a number of moves equal to or under 'Par'.

 Undo & Reset

Hit the undo button toundo a move. Hit the reset to start over Clear Game Data Use this to delete all progress and set the app back to when it was first used. Warning! this cannot be undone. You will lose all your progress.
